# Hospedagem-de-Site-Est-tico-na-AWS-com-Amazon-S3-
Este projeto demonstra como criar e implantar um site estático utilizando o Amazon S3, IAM e a AWS CLI, seguindo as boas práticas de automação e controle de acesso na AWS.  O laboratório simula a hospedagem de um site simples (cafeteria e padaria), tornando-o acessível publicamente via endpoint de site estático do S3.
